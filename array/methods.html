Array方法

concat push unshift shift pop splice(index,num,....) reverse join slice forEach indexOf sort map filter some every reduce find findIndex includes
  • concat 连接两个或者多个数组，并返回新数组
  • push 在数组末尾新增一个或者多个元素，并返回新的长度
  • unshift 在数组开头新增一个或者多个元素，并返回新的长度
  • pop 删除并返回数组最后一个元素
  • shift 删除并返回数组第一个元素
  • splice(index,num,....) 指定数组下标删除/新增/代替
  • forEach(function(curValue,index,arr){},thisValue) curValue当前元素项，index下标、arr原数组，thisValue默认指向window
    forEach无法提前结束、会跳过空值、没有返回值、循环次数由数组初始长度决定，只适合集合和遍历
    参考
  • reverse 没有参数 颠倒排序 改变原数组并返回新数组
  • sort(callback) 修改原数组 默认unicode排序
    function(a,b){return a-b}顺叙
    function(a,b){return b-a}倒叙
    function(a,b){return 0.5 - Math.random()} 乱序
    function(a,b){return a.c - a.c} 对象排序
  • join(string) 按照string方式拼接数组元素成为字符串
  • indexOf(item,start) item查找元素 start指定下标 可负 找不到返回-1 返回找到第一个匹配元素下标
  • lastIndexOf(item,start) item查找元素 start指定下标 可负 找不到返回-1 返回找到最后一个匹配元素下标
  • slice(start,end) 剪切数组 返回新数组 start 可负
  • map(function(curValue,index,arr){},thisValue) curValue当前元素项，index下标、arr原数组，thisValue默认指向window
    map效果跟forEach一样，但是唯一区别是map返回新数组，且性能比forEach高70%左右
  • map、filter、some、every、reduce、find 无法提前结束、会跳过空值
  • filter/some/every(function(curValue,index,arr), thisValue)
  • reduce(function(total,curValue,index,arr), initialValue)

push、pop、unshift、shift、splice、reverse、sort 改变原数组，其他都是返回新数组
