修饰符

  • i => ignore 忽视大小写
  • g => global 全局查找
  • m => multiple 多次匹配

元字符

  • . 查找单个字符，除换行符和行结束符
  • \w 查找单词字符 等价于[0-9A-Za-z_]
  • \W 查找非单词字符 等价于[^0-9A-Za-z_]
  • \d 查找数字 等价于[0-9]
  • \D 查找非数字 等价于[^0-9]
  • \b 查找单词边界
  • \B 查找非单词边界
  • \0 查找NULL字符
  • \n 查找换行符
  • \f 查找换页符

描述字符范围

  • [abz] 查找方括号内任意一个字符。
  • [^abz] 查找方括号内任意一个字符。
  • [0-9] === \d 查找数字。
  • [^0-9] === \D 查找非数字。
  • (x | y) 查找任何以 | 分隔的选项。
  • [a-z] 查找a-z字符
  • [A-z] 查找大小写字符

量词

  • n+ 等价于{1, } 匹配任何包含至少一个 n 的字符串。
  • n? 等价于{0, 1} 匹配任何包含零个或一个n的字符串
  • n* 等价于{0, } 匹配任何包含零个或多个n的字符串
  • {n} 匹配前一项n次
  • {n, m} 至少n次，最多m次
  • {n, } 至少n次

惰性匹配 ?

?, {n}, {n, m} 弱贪婪性 +, *, {n, } 强贪婪性

  • ?? 尽量匹配，但是为了满足限定条件也可能最多匹配 1 次，相当于 {0,1}?。
  • +? 尽量匹配 1 次，但是为了满足限定条件也可能匹配任意次，相当于 {1,}?
  • *? 尽量不匹配，但是为了满足限定条件也可能匹配任意次，相当于 {0,}?。
  • {n}? 尽量匹配 n 次。
  • {n, m}? 尽量匹配 n 次，但是为了满足限定条件也可能最多重复 m 次
  • {n, } 尽量匹配 n 次，但是为了满足限定条件也可能匹配任意次

分组

$1 $2 ... $n 对应分组内容 \1 \2 反向引用 分组详细介绍

  • () 捕获型
  • (a | b) 选择
  • (?:) 非捕获型
  • (?=) 正向前瞻型
  • (?!) 反向前瞻型

正则的创建方法

  • 隐式创建 let reg = /\d+/
  • 显式创建 new RegExp(正则表达式,"gi")

正则对象的方法

  • test() 返回 true/false
  • exec() 返回一个数组，其中存放匹配的结果。如果未找到匹配，则返回值为 null。

支持正则表达式的string对象的方法

  • search 返回第一个以regexp相匹配的起始位置，没有返回-1
  • match 在字符串内检索指定的值，找到一个或多个正则表达式的匹配。 以数组的形式返回相匹配的内容，返回匹配结果的数组，数组内容依赖于regexp是否具有全局标志g
  • replace 用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。
  • split 用于把一个字符串分割成字符串数组
